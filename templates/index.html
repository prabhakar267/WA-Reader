Welcome to WA Reader

WA Reader is a web-tool aimed to help users view their WhatsApp back-up chats in a familiar user-interface.
You can upload the text file of your chat to the tool and you will be able to view instantly.
The tool is completely open-sourced making it secure to use.
You can upload your WhatsApp Text file which is of format .txt that you get from the Email Conversation button in your WhatsApp app.

We recently, launched the support for uploading the .zip file of your chat if you wish to view the attachments as well. This feature is still in development, please report bugs if you face any. Since we have to show the media files they are temporarily stored on the server, but as soon as you refresh the page, or click on view another chat all the files get deleted.

Ads on the website helps in covering the hosting costs.
If WA-Reader helped you read chats of someone special, consider helping out the developers by contributing here.

Understanding your WhatsApp chats: Your WhatsApp messages are automatically backed up and saved daily to your phone's memory depending on your settings and preferences. If you uninstall WhatsApp from your phone, but don't want to lose any of your messages, be sure to manually back up or export your chats before uninstalling. For other queries, please refer to the Frequently Asked Questions (FAQs) below
I keep getting "{{ data['default_error_message'] }}", even though I am uploading a file?
It happens since each mobile device and WhatsApp version has it's own way of storing the chats. You can open a new issue on the GitHub repository and share a small piece of chat (if possible) so that the viewer can be updated according to the formatting of your device and WhatsApp version.
Do mention your WhatsApp version, mobile OS and device name for better tracking.
Is my chat getting saved?
No.
Your chat gets deleted as soon as you upload the file. You can feel safe on that front.
Can I preview my chat if I have uploaded it once?
No.
Since your chat is getting deleted as soon as you upload the File, you cannot preserve your chat history for you to visit later.
Are there any limitations to what can be backed up from the app?
Some of the limitations mentioned on the WhatsApp FAQs are:
  • If you choose to attach media, the most recent media sent will be added as attachments.
  • When exporting with media, you can send up to 10,000 latest messages. Without media, you can send 40,000 messages. These constraints are due to maximum email sizes.

Note from developers: WA Reader was initially developed to read a personal chat and it has evolved since then. There can be some issues with the chats since the format with which WhatsApp exports chats can change anytime without any notice. Please feel free to open a new issue on GitHub if you are not able to use the app properly. You can find the source code on GitHub here which has instructions for you to even set up the code locally. Shout out to all the contributors who helped shape the project to its current form.
